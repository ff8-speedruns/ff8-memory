Final Fantasy 8

Welcome to Brofar's collection of information, resources, and tools for Final Fantasy VIII PC speedrunning. The information here has been sourced from many people and websites (including my own research) as seen in the credits section.

List Discord Credits

Projects

Name Description Status Link
Fish Fin Manip Table lookup for the fish fin manipulation. Working Open
Encounter Calculator Using the previous encounter formation ID along with the encounter counter (total number of encounters) we can tell which encounter formation we will receive next. Working Open
Cheat Engine Tables (PC) A file for use with Cheat Engine to read various aspects of the game's memory. Always WIP Open
Caraway Code Reads a specific memory address from FF8_EN or FF8_FR (Steam 2013 / non-remaster) containing the code to get into Caraway's mansion. Working Open
Controls Randomizer Randomizes controller inputs while playing FF8 by reading and shuffling the control mapping in the game's memory while you play. Has some configuration options. Working Open
Card RNG Manip Tool A webpage to keep track of early game RNG events to help with Zell card. In Progress Open
Final Party Manipulation Conversion of Pingval's final party manipulation code for Playstation and awesomeWaves' modifications for PC ruby script to JS, so that it can be run on the web without requiring runners to install Ruby. In Progress Open
